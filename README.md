# Bot de Notificação de Reunião 📅

## Descrição
Este bot tem como objetivo enviar mensagens para todos os membros de um grupo, informando sobre reuniões, eventos ou atualizações importantes. Uma ferramenta simples e eficaz para garantir que todos estejam cientes das próximas atividades.

## Funcionalidades
- 📩 Envio automático de mensagens para todos os membros do grupo
- ⏰ Agendamento de reuniões com lembretes
- 🔔 Notificações personalizáveis
- 🗂️ Armazenamento de informações sobre reuniões passadas
